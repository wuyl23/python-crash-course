**相关方法**

* `append()` 将元素添加到列表末尾
* `insert()` 在列表任何位置添加新元素
* `del` 删除指定位置的元素
* `pop()` 弹出指定位置的元素
* `remove()` 删除第一个指定的值
* `sort()` 排序
* `reverse()` 倒序
* `len()` 获取长度
